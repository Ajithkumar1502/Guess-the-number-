About This Project

More Than a Guessing Game

Guess The Number Pro is a showcase of modern, framework-free frontend development. Built with pure HTML5, CSS3, and Vanilla JavaScript, every interaction — from the glassmorphic UI to the confetti celebration — is handcrafted without libraries. It demonstrates persistent state management via localStorage, modular JavaScript architecture, and responsive, accessible design that works on every device.

HTML5 CSS3 Vanilla JS localStorage CSS Animations ARIA Accessibility Responsive Design Glassmorphism
What's Inside

Built to Impress

🎯

Smart Difficulty

Three calibrated modes — Easy (1–50), Medium (1–100), Hard (1–500) — each with its own scoring multiplier.

🏆

Leaderboard

Persistent top-10 rankings per difficulty, stored locally. Named entries, automatic sorting, filterable tabs.

📊

Stats Dashboard

Tracks win rate, average attempts, best score, and games played — all persisted across sessions.

🎉

Win Celebration

A canvas-based confetti burst and animated victory modal every time you crack the code.

🌙

Dark / Light Mode

Full theme system with smooth CSS transitions. Preference saved in localStorage automatically.

♿

Accessible by Design

Full keyboard navigation, ARIA labels throughout, visible focus indicators, and WCAG-compliant contrast.
